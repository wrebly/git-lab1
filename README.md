# Tandoor Kitchen (Система управління рестораном)
Full-stack вебсайт ресторану із системою бронювання столиків та панеллю адміністратора.


Опис ідеї MVP

Проєкт призначений для автоматизації взаємодії між клієнтом та рестораном. Основним функціоналом (MVP) є:

    Бронювання: Можливість вибору дати, часу та кількості осіб через веб-інтерфейс.

    Меню: Відображення переліку страв з актуальними цінами.

    Адмін-панель: Перегляд списку поточних бронювань та керування статусами столиків.

### Стек технологій
* HTML, CSS, JavaScript
* Node.js (Backend)
